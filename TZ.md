# Техническое задание на проект: онлайн-магазин шрифтов с настраиваемыми лицензиями.

## 1. Цель проекта
Создать концепт онлайн магазина шрифтов с предпросмотром начертаний, конфигуратором лицензий и оформлением заказа. Сервис упрощает покупку шрифтов и снижает потребность в ручной коммуникации с менеджерами за счет понятной конфигурации лицензий и автоматизированного оформления заказа. Административная часть обеспечивает управление каталогом, начертаниями и ценами лицензий. Пользовательская часть обеспечивает просмотр шрифта, выбор лицензии, создание корзины и оформление заказа.

## 2. Роли пользователей
- **Гость**
  - Просмотр каталога и карточек шрифтов
  - Просмотр доступных начертаний и предпросмотра
  - Сбор корзины (анонимная корзина по UUID)

- **Авторизованный пользователь**
  - Все возможности гостя
  - Сохранение корзины в аккаунт
  - Оформление заказа
  - Просмотр истории заказов

- **Администратор**
  - CRUD для шрифтов, начертаний, связок "шрифт + начертание"
  - Управление ценами по типам лицензий
  - Просмотр заказов и состава заказов
  - Управление пользователями (активация, блокировка)

## 3. Модели данных
- **Модель 1: Font**
  - Поля: `name`, `author` (nullable), `date_release` (nullable), `desc`

- **Модель 2: FontFace**
  - Поля: `font` (ForeignKey -> Font), `style` (ForeignKey -> FontStyle)
  - Ограничения: уникальность пары (`font`, `style`)

- **Модель 3: FontFacePrice**
  - Поля: `face` (ForeignKey -> FontFace), `license_type` (choices), `price`, `currency`

- **Модель 4: Cart**
  - Поля: `id` (UUID), `user` (nullable), `items` (ManyToMany -> FontFacePrice), `sum`

- **Модель 5: Order**
  - Поля: `number` (UUID), `user` (nullable), `created_at`

- **Модель 6: OrderItem**
  - Поля: `order` (ForeignKey -> Order), `font_face_with_price` (ForeignKey -> FontFacePrice)
  - Ограничения: уникальность пары (`order`, `font_face_with_price`)

- **Модель 7: CustomUser**
  - Поля: `email` (unique), `username`, `is_active`

## 4. Ключевой функционал
- Просмотр каталога шрифтов и переход в карточку выбранного шрифта
- Предпросмотр начертаний и выбор конкретного начертания (FontFace)
- Выбор типа лицензии (LicenseType) и отображение итоговой цены (FontFacePrice)
- Добавление выбранной конфигурации в корзину, изменение состава корзины, пересчет суммы
- Оформление заказа с созданием `Order` и `OrderItem`
- Просмотр истории заказов авторизованным пользователем
- Администрирование каталога, начертаний и цен лицензий

## 5. Внешние интеграции / Аналитика
- **Аналитика:** библиотека **Pandas**
  - Расчеты: топ шрифтов по покупкам, популярность лицензий по типам (на основе данных заказов)
